### 증분 정적 재생성
```
정적 페이지 생성(SSG)은 미리 만들어 놓은 페이지를 서비스 하기 때문에 속도는 빠르지만,
한번 생성하고 나면 수정이 불가능함. 이러한 단점을 보완하고자 나온 것이 증분 정적 재생성
(ISR)임. 이미 생성된 페이지를 일정 시간이 지난 후에 다시 생성함.
```
